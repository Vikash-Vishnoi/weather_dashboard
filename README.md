# Weather Dashboard

🌍 **Live:** [https://weather-dashboard-alpha-kohl.vercel.app/](https://weather-dashboard-alpha-kohl.vercel.app/)

A real-time weather dashboard built with Next.js, providing current weather and a 5-day forecast. Powered by the [OpenWeatherMap API](https://openweathermap.org/).

## 🚀 Setup

1. **Clone & Install:**
   ```bash
   git clone https://github.com/Vikash-Vishnoi/weather_dashboard.git && cd weather_dashboard
   npm install
   ```

2. **Environment:**
   Create `.env.local` and add your API key:
   ```
   WEATHER_API_KEY=your_api_key_here
   ```

3. **Run:**
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000)

## 🛠 Tech Stack

* Next.js
* Tailwind CSS
* Lucide Icons
* [OpenWeatherMap API](https://openweathermap.org/)
